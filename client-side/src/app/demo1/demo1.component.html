Demo 1: Enable Angular to use custom elements.

To use any Cubbles component it is required to add CUSTOM_ELEMENTS_SCHEMA to the @NgModule, allowing custom HTML Elements to be used. Additionally define a window.cubx.CRCInit object to declare related dependencies on component level.
