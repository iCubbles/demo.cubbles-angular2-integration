Demo 1

This demo shows how to easily integrate Cubbles into an existing Angular 2 Application. The project is build with the angular-cli and uses the Cubbles Runtime Extension (source on github)

To use any Cubbles component it is required to add CUSTOM_ELEMENTS_SCHEMA to the @NgModule, allowing custom HTML Elements to be used.
